SQL Lessons

Learn SQL step by step with detailed explanations and practical examples.

Lesson 1: Introduction to SQL

SQL (Structured Query Language) is a standard language for storing, manipulating, and retrieving data in databases. It's used to communicate with relational database management systems (RDBMS) like MySQL, PostgreSQL, SQLite, and SQL Server.

Key Concepts:

  • Database: A collection of organized data
  • Table: A structure that organizes data into rows and columns
  • Row: A single record in a table
  • Column: A field in a table that represents a specific attribute

Lesson 2: SELECT Statement

The SELECT statement is used to query data from a database. It's the most commonly used SQL command.

Basic Syntax:

SELECT column1, column2, ... FROM table_name;

Examples:

-- Select all columns from students table
SELECT * FROM students;

-- Select specific columns
SELECT name, age FROM students;

Lesson 3: WHERE Clause

The WHERE clause is used to filter records based on specific conditions.

Basic Syntax:

SELECT column1, column2, ... FROM table_name WHERE condition;

Examples:

-- Select students older than 20
SELECT * FROM students WHERE age > 20;

-- Select students with specific name
SELECT * FROM students WHERE name = 'John';

Common Operators:

  • = Equal
  • != or <> Not equal
  • > Greater than
  • < Less than
  • >= Greater than or equal
  • <= Less than or equal
  • LIKE Pattern matching
  • IN Match any value in a list

Lesson 4: ORDER BY Clause

The ORDER BY clause is used to sort the result set in ascending or descending order.

Basic Syntax:

SELECT column1, column2, ... FROM table_name ORDER BY column1 [ASC|DESC];

Examples:

-- Sort students by age in ascending order
SELECT * FROM students ORDER BY age ASC;

-- Sort students by name in descending order
SELECT * FROM students ORDER BY name DESC;

-- Sort by multiple columns
SELECT * FROM students ORDER BY grade DESC, age ASC;

Lesson 5: INSERT Statement

The INSERT statement is used to add new records to a table.

Basic Syntax:

INSERT INTO table_name (column1, column2, ...) VALUES (value1, value2, ...);

Examples:

-- Insert a new student
INSERT INTO students (name, age, grade) VALUES ('Alice', 22, 'A');

-- Insert multiple students
INSERT INTO students (name, age, grade) VALUES
('Bob', 21, 'B'),
('Charlie', 23, 'A');

Lesson 6: UPDATE Statement

The UPDATE statement is used to modify existing records in a table.

Basic Syntax:

UPDATE table_name SET column1 = value1, column2 = value2, ... WHERE condition;

Examples:

-- Update a student's grade
UPDATE students SET grade = 'A+' WHERE name = 'Alice';

-- Update multiple columns
UPDATE students SET age = 24, grade = 'B+' WHERE name = 'Bob';

Warning: Always use a WHERE clause with UPDATE to avoid updating all records!

Lesson 7: DELETE Statement

The DELETE statement is used to remove records from a table.

Basic Syntax:

DELETE FROM table_name WHERE condition;

Examples:

-- Delete a specific student
DELETE FROM students WHERE name = 'Charlie';

-- Delete students with low grades
DELETE FROM students WHERE grade = 'F';

Warning: Always use a WHERE clause with DELETE to avoid deleting all records!

Lesson 8: JOINS

JOINs are used to combine rows from two or more tables based on a related column between them.

Types of JOINs:

  • INNER JOIN: Returns records that have matching values in both tables
  • LEFT JOIN: Returns all records from the left table and matched records from the right table
  • RIGHT JOIN: Returns all records from the right table and matched records from the left table
  • FULL OUTER JOIN: Returns all records when there is a match in either left or right table

Basic Syntax:

SELECT columns FROM table1
INNER JOIN table2 ON table1.column = table2.column;

Example:

-- Join students with courses
SELECT students.name, courses.course_name
FROM students
INNER JOIN enrollments ON students.id = enrollments.student_id
INNER JOIN courses ON enrollments.course_id = courses.id;

Lesson 9: GROUP BY Clause

The GROUP BY clause groups rows that have the same values into summary rows. It's often used with aggregate functions.

Basic Syntax:

SELECT column1, aggregate_function(column2)
FROM table_name
GROUP BY column1;

Common Aggregate Functions:

  • COUNT() - Count number of rows
  • SUM() - Sum of values
  • AVG() - Average of values
  • MAX() - Maximum value
  • MIN() - Minimum value

Examples:

-- Count students by grade
SELECT grade, COUNT(*) as student_count
FROM students
GROUP BY grade;

-- Average age by grade
SELECT grade, AVG(age) as average_age
FROM students
GROUP BY grade;

Lesson 10: HAVING Clause

The HAVING clause is used to filter groups created by GROUP BY. It's like WHERE but for groups.

Basic Syntax:

SELECT column1, aggregate_function(column2)
FROM table_name
GROUP BY column1
HAVING condition;

Example:

-- Show grades with more than 2 students
SELECT grade, COUNT(*) as student_count
FROM students
GROUP BY grade
HAVING COUNT(*) > 2;
